Technical Toolbox

Apart from learning the basics of web fundementals regarding HTML5, CSS3, JavaScript, and JQuery, these are the three full stacks I've learned (so far):

Python

Python was the first stack I adopted during my time at the Coding Dojo. From this, I was introduced to object-oriented programming and its concepts making your code more organized and modular. After being introduced to the architectual frameworks, I further on equiped and built my knowledge using Flask and Django. I was finally able to create my own applications by using MySQLWorkbench, writing SQL queries, and able to deploy them on AWS .

Ruby on Rails

After experiencing the process of learning a full stack, it quickly minimized the difficulty in learning my next stack, Ruby on Rails. I was able to practice and fine-tuned writing queries using Active Record to take care of my database and understood the advantages for building quick prototypes. In my journey using the pure object-oriented language, I learned using RSpec and Capybara. Lastly, I learned how to deploy my projects with PostgreSQL .

MEAN

Finally, the last thing I learned at the Dojo was the MEAN (MongoDB, Express, Angular and Node) stack. I got the chance to experience working with the database using MongoDB, node framework with Express, javascript/typescript framework through Angular, and lastly with Node.
